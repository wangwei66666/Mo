###Jetpack

- 遵循最佳做法
  - Android Jetpack组件采用最新的设计方法构建，具有向后兼容性，可以减少崩溃和内存泄漏。
- 取消样板代码
  - Android Jetpack可以管理各种繁琐的Activity（如后台任务、导航和生命周期管理），以便您专注于打造出色的应用。
- 减少不一致
  - 这些库可在各种Android版本和设备中以一致的方式运作，助您降低复杂性。
